# Province Flag Files

Put all province flags in this folder.

The app reads these exact filenames:

```text
central.png
chimbu.svg
east-new-britain.png
east-sepik.png
eastern-highlands.svg
enga.png
gulf.png
hela.png
jiwaka.png
madang.png
manus.png
milne-bay.png
morobe.png
national-capital.png
new-ireland.png
north-solomons.svg
oro.png
sandaun uses west-sepik.png
southern-highlands.png
west-new-britain.png
western.jpg
western-highlands.svg
```

If your downloaded file has another extension, either rename it to match the list above or update `src/data/province-media.ts`.

The old dance media folder is still used for videos and GIFs:

```text
public/media/provinces/central/dance.mp4
public/media/provinces/central/dance.gif
```
